# Sistema de recomendación de películas (MovieLens)

Proyecto desarrollado en Python a partir del dataset MovieLens para la construcción de un sistema de recomendación de películas. El trabajo implementa filtrado colaborativo mediante SVD, recomendación basada en contenido con TF-IDF y un enfoque híbrido que combina ambos métodos.

El notebook incluye el desarrollo paso a paso, con explicaciones detalladas del proceso, desde la carga y preparación de los datos hasta la generación y evaluación de las recomendaciones. El enunciado del ejercicio define los objetivos y el alcance del proyecto.

## Librerias
- pandas, numpy
- scikit-learn
- matplotlib
