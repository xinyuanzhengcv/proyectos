# Portfolio – Big Data & Data Engineering

Este repositorio recoge una selección de proyectos desarrollados durante el **Máster en Big Data & Data Engineering**, orientados a demostrar competencias técnicas y conceptuales de la ingenieria de datos en entorno de big data. 

Los proyectos cubren distintas áreas clave: programación avanzada, procesamiento de datos, ETL, bases de datos y diseño de arquitecturas de datos.

---
## 1. Python Avanzado – Análisis de Multas de Tráfico (traficFines)

**Tecnologías y librerías**  
Python, pytest, cobertura de tests, Jupyter Notebook, empaquetado de librerías.

**Descripción**  
Desarrollo de un paquete Python para `descargar`, `cachear` y `analizar datos` abiertos de multas de circulación del Ayuntamiento de Madrid. El proyecto pone el foco en la calidad del código, la testabilidad y la organización modular.

**Puntos clave**  
- Diseño modular con separación entre caché y lógica de análisis.  
- Tests unitarios con simulación de datos (sin acceder a URLs reales usando caché).  
- Validación funcional mediante notebooks con datos reales (2017–2025).  
- Buenas prácticas de empaquetado y cobertura de código.

---
## 2. Scala – ETL de Datos

**Tecnologías y librerías**  
Scala, SBT, Typesafe Config.

**Descripción**  
Implementación de un proceso ETL en Scala que lee múltiples ficheros CSV desde un directorio, realiza agregaciones configurables y genera un CSV de salida utilizando configuración externa.

**Puntos clave**  
- Separación clara de responsabilidades (lectura, parsing, agregación y escritura).  
- Configuración externa sin necesidad de recompilar.  
- Manejo de errores, líneas inválidas y múltiples formatos de fecha.  
- Arquitectura extensible para nuevos modos de agregación o claves de agrupación.

---
## 3. Arquitectura de Datos

**Tecnologías / Enfoque**  
Diseño de arquitecturas de datos, documentación técnica, diagramas.

**Descripción**  
Proyecto académico centrado en el diseño y análisis de una arquitectura de datos completa, desde la ingesta hasta la capa de consumo.

**Puntos clave**  
- Análisis de flujos de ingesta, almacenamiento, procesamiento y explotación.  
- Evaluación de decisiones técnicas y su impacto en escalabilidad y mantenimiento.  
- Uso de diagramas arquitectónicos y documentación detallada.  
- Informe final en PDF con conclusiones técnicas.

---

## 4. Bases de Datos – MySQL

**Tecnologías y librerías**  
MySQL, SQL.

**Descripción**  
Conjunto de scripts SQL para la creación y gestión de una base de datos relacional.

**Puntos clave**  
- Creación de tablas y modelado básico de datos.  
- Inserción de datos y consultas SQL fundamentales.  
- Enfoque práctico en manipulación y consulta de datos.
